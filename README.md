# Healthcare Analytics Dashboard

## Project Overview
This project focuses on analyzing healthcare data to monitor patient trends, appointments, revenue, and operational performance. The dashboard helps healthcare stakeholders make data-driven decisions through interactive visualizations.

## Tools Used
- Power BI
- SQL
- Excel
- Power Query
- DAX
- Tableau

## Business Objectives
- Track patient growth and appointments
- Monitor revenue and collections
- Measure doctor utilization
- Analyze operational KPIs
- Improve healthcare decision-making

## Key KPIs
- Total Patients
- Total Appointments
- Total Revenue
- Collection Amount
- Doctor Utilization Rate
- Patient Growth %
- Department Performance

## My Role
- Extracted data using SQL
- Performed data cleaning using Power Query
- Built data models and relationships
- Created DAX measures and KPIs
- Developed interactive dashboards
- Generated business insights for stakeholders
